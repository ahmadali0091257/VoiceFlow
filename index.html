How to Use VoiceFlow AI

Get started in minutes with these simple steps

Download & Extract

Download the full project folder from the link below and extract it to your PC.

Open in VS Code

Open the extracted folder in Visual Studio Code or your preferred code editor.

Install Requirements

Open terminal and run:

pip install -r requirements.txt

Install PySide6

After requirements installation completes, run:

pip install PySide6

Launch the App

Run the main application file:

python main.py

The app will launch and be ready to use!

Configure APIs

Add your API keys:

  • DeepSeek AI API
  • Groq API (Free)

🎤 How to Use Voice Dictation

Step 1: Open any application where you want to dictate (code editor, document, etc.)
Step 2: Press and HOLD the shortcut keys:
CTRL + WINDOWS
Step 3: Start speaking clearly. The app will capture your voice in real-time.
Step 4: When you're done speaking, RELEASE the keys. Your text will appear automatically!

✨ Works in any app - even those without built-in voice options!
