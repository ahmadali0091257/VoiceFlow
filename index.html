Frequently Asked Questions

Got questions? We've got answers.

Is VoiceFlow AI completely free to use?
+
Yes! VoiceFlow AI is 100% free and open-source. You can use it without any limitations, subscriptions, or hidden fees. When using Groq API (which has a generous free tier), you can dictate without any cost. If you choose to use DeepSeek AI, you'll need an API key, but the software itself remains free.
Does it work offline?
+
VoiceFlow AI requires an internet connection for the AI transcription service (Groq or DeepSeek). However, the application itself runs locally on your machine, and your audio is processed securely. We're working on offline capabilities for future versions.
What languages are supported?
+
VoiceFlow AI supports multiple languages through the AI backends. Groq and DeepSeek both support English, Spanish, French, German, Chinese, Japanese, and many more. The exact language support depends on which API you choose to use.
Is my voice data private and secure?
+
Absolutely! Your privacy is our top priority. VoiceFlow AI runs locally on your computer, and we don't collect or store any of your voice data. The audio is sent directly to your chosen AI provider (Groq or DeepSeek) for transcription, and you control the API keys. No third-party tracking or data collection.
Can I use it in any application?
+
Yes! That's one of the key features of VoiceFlow AI. It works in any application that accepts text input - VS Code, Microsoft Word, Google Docs, Notepad, email clients, web browsers, chat applications, and more. Just press Ctrl + Win, speak, and your text appears wherever your cursor is.
How accurate is the transcription?
+
The accuracy depends on the AI backend you choose. Both Groq and DeepSeek offer high accuracy rates (typically 95%+ for clear speech in quiet environments). Accuracy can vary based on your microphone quality, background noise, accent, and speaking clarity. For best results, use a good microphone and speak clearly.
Do I need programming knowledge to set it up?
+
Basic familiarity with command line is helpful, but the setup process is straightforward. You just need to: 1) Download and extract the folder, 2) Run two pip install commands, 3) Run python main.py. The entire process takes less than 5 minutes. Detailed instructions are provided on this page.
Can I contribute to the project?
+
Yes! VoiceFlow AI is open-source, and we welcome contributions from the community. Whether it's bug fixes, new features, documentation improvements, or translations, your help is appreciated. Check out the GitHub repository to get started.

Quick Setup Guide

Get started in under 5 minutes

1

Download Project

Download the complete project folder from Google Drive and extract it to your desired location.

2

Open in VS Code

Open the extracted folder in Visual Studio Code or your preferred code editor.

3

Install Dependencies

Open terminal and run:

pip install -r requirements.txt
4

Install PySide6

Install the GUI framework:

pip install PySide6
5

Launch Application

Start the app with:

python main.py
6

Configure APIs

Add your API keys:

  • DeepSeek AI API (optional)
  • Groq API (free tier available)

How to Use

Simple voice dictation in 4 steps

Step 1 Open any application where you want to dictate text (code editor, document, email, etc.)
Step 2 Press and hold Ctrl + Win keys together
Step 3 Speak clearly. Your voice is converted to text in real-time as you talk.
Step 4 Release the keys when done. Your transcribed text appears automatically!

✨ Works in any text field—even apps without built-in voice support
