Installation

Get up and running in under 5 minutes.

1

Download & Extract

Download the project folder and extract it to your desired location.

2

Open in VS Code

Open the extracted folder in Visual Studio Code or your preferred editor.

3

Install Dependencies

Open terminal and run:

pip install -r requirements.txt
4

Install PySide6

Run the following command:

pip install PySide6
5

Launch the App

Start the application:

python main.py
6

Add API Keys

Configure your API keys:

  • DeepSeek AI API
  • Groq API (free tier available)

How to Use

Step 1 Open any application where you want to dictate text
Step 2 Press and hold Ctrl + Win
Step 3 Speak clearly. Your voice is converted to text in real-time.
Step 4 Release the keys when you're done. Text appears automatically.

Works in any text field—even apps without built-in voice support.
